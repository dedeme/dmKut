Creación

Los párrafos tienen que estar separados por dos retornos de carro.

Esta línea y esta están en el mismo párrafo. Esta otra está en párrafo distinto.

Produce:

Esta línea y esta están en el mismo párrafo.

Esta otra está en párrafo distinto.

Corte de línea

Dentro del párrafo se puede forzar que una linea se corte en el punto deseado, terminándola con '\'.

Esta línea \ y esta están en el mismo párrafo; pero en distinta línea. Esta otra está en párrafo distinto.

Produce:

Esta línea
y esta están en el mismo párrafo; pero en distinta línea.

Esta otra está en párrafo distinto.

Formato

Se pueden utilizar '*' y '_' para establecer texto con formato '<code>' y 'cursiva' respectivamente.

Estos signos deben aparecer antes y despues del texto a resaltar.

Si se requiere insertar los propios caracteres '*' o '_' se debe escribir '**' o '__' repectivamente.

También

La siguiente palabra está en _cursiva_; pero esta otra no: __cursiva__. Lo mismo es aplicable a *code* y **code**.

Produce:

La siguiente palabra está en cursiva; pero esta otra no: _cursiva_. Lo mismo es aplicable a code y *code*.

Inyección de carácter.

Cuando se necesita usar un carácter que es interpretado como un caracter especial (p.ej. *), se puede escribir usando la composición '\[caracter]'.

Por ejemplo:

Se consigue que *\[*]los asteriscos \[*]* aparezcan como tales

produce

Se consigue que *los asteriscos * aparezcan como tales

Inyección de espacio en blanco.

Cuando se requiere que aparezcan blancos que son eliminados por el código HTML, se puede utilizar el punto elevado '·' (no confundir con el punto normal '.').

Si se requiere el propio punto elevado se puede utilizar \[·].

Ejemplo:

Se precisa sangrado para \ ·· Esto y \ ·· Esto Pero no para \ \[·]\[·]Esto ni \ \[·]\[·]Esto

Produce:

Se precisa sangrado para
   Esto y
   Esto

Pero no para
··Esto ni
··Esto

Inyección de HTML

Una inyección de HTML se puede realizar en el interior de un párrafo o a su comienzo.

Interior en un párrafo.

Se introduce y se finaliza con ``.

Este texto está en &lt;b>formato &lt;big>normal&lt;/big>&lt;/b>; ``pero este lo esta en &lt;b>formato &lt;big>de inyección&lt;/big>&lt;/b>``.

Produce:

Este texto está en <b>formato <big>normal</big></b>; pero este lo esta en formato de inyección.

Al comienzo del párrafo.

En este caso es conveniente colocar los caracteres `` iniciales en una línea aparte, para evitar interferencia con la opción de presentación de código.

Este texto está en &lt;b>formato &lt;big>normal&lt;/big>&lt;/b>. `` ¨ Pero este lo esta en &lt;b>formato &lt;big>de inyección&lt;/big>&lt;/b>``.

Produce:

Este texto está en <b>formato <big>normal</big></b>.

Pero este lo esta en formato de inyección.
