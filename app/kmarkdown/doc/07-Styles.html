Kmarkdown utiliza ciertos elementos que han de colocarse en la hoja de estilos o en el encabezado.

TABLE.kmd-main { width : 100% } TD.kmd-fm { border: 1px solid rgb(110,130,150); background-color: rgb(255, 250, 250);padding: 4px; } TD.kmd-fm2 { border: 1px solid rgb(110,130,150); padding: 4px; } SPAN.kmd-title { color:#002040; font-size:18px; } SPAN.kmd-fm { background-color: rgb(255, 250, 250); border: 1px solid rgb(110,130,150); padding: 4px;border-radius: 4px; } IMG.kmd-fm { border: 1px solid rgb(110,130,150); } DIV.kmd-aceEditor{ width: 650px; height: 0px; border-radius: 4px; border: 1px solid rgb(0,0,0); }
