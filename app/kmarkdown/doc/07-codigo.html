Creación

Se pueden presentar recuadros con código utilizando la siguiente sintaxis:

``lenguage
código
``
Código en javascript: ``javascript function sum3(a) { return a + 3; } ``

Produce:

Código en javascript:

function sum3(a) { return a + 3; }

NOTA:

Escribir texto en la misma línea detrás de las marcas de cierre `` es un error.

Esto es un error:

Código en javascript: ``javascript function sum3(a) { return a + 3; } `` Esto es un error

Requisitos

Para que los cuadros de código funciones es necesario:

  1. Tener instalado el editor Ace.

  2. Que el editor tenga el módulo del lenguage correspondiente.

Instalación

Es conveniente descargar la versión src-noconflict o src-min-noconflict de https://github.com/ajaxorg/ace-builds.

Por defecto este directorio se debe renombrar como "ace" y debe estar en el mismo directorio que los archivos .kmd. Para ello se pueden usar enlaces simbólicos.

Es posible cambiar la ubicación del directorio manipulando el archivo generado automáticamente por kmarkdown 000header.html.

Lenguaje

Los lenguajes disponibles se encuentra en el dírectorio "ace" con nombres tipo mode-lenguaje.js. Por ejemplo mode-c_cpp.js se corresponde con los lenguajes 'c' y 'c++', mode-javascript-js con el lenguaje 'javascript', etc.
